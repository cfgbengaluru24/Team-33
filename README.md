# Team-33

## Prerequisites

Make sure you have Node.js and npm installed on your system.

## Getting Started

### Admin

1. **Navigate to the Admin Directory**:

   ```sh
   cd admin
   ```

2. **Setup Frontend**:

   ```sh
   cd frontend
   npm install
   npm run dev
   ```

3. **Setup Backend**:
   ```sh
   cd ../backend
   npm install
   npm start
   ```

### Trainer

1. **Navigate to the Trainer Directory**:

   ```sh
   cd trainer
   ```

2. **Setup Project**:
   ```sh
   npm install
   npm run dev
   ```

### Trainee

1. **Navigate to the Trainee Directory**:

   ```sh
   cd trainee
   ```

2. **Setup Frontend**:

   ```sh
   cd frontend
   npm install
   npm run dev
   ```

3. **Setup Backend**:
   ```sh
   cd ../backend
   npm install
   node server.js
   ```

## Running the Projects

- **Admin**:

  - Frontend: `npm run dev`
  - Backend: `npm start`

- **Trainer**:

  - Run Project: `npm run dev`

- **Trainee**:
  - Frontend: `npm run dev`
  - Backend: `node server.js`
